# chmod

The chmod command in Linux uses options, user classes, operators, and permissions to manage file access control. 

Permissions can be set using two modes: symbolic (letters and symbols) and numeric (octal numbers). 

## options
https://en.wikipedia.org/wiki/Chmod

## example
`chmod 755 script.sh`   
Grants the owner read, write, and execute permissions, and the group and others read and execute permissions.

`chmod 600 filename`   
Only the owner can read and write the file.
